Configure Services

Data Applications

  • Manage Applications

User Management

  • Invite Users
    Send invites to new users so they can view your Applications and Dashboards.

  • Manage Users
    Grant users access to Applications. Edit/Delete users.

  • Manage Roles
    Fathym uses roles to control what users can and cannot do within your account.
